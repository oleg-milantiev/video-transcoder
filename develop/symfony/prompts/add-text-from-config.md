Работа с vue frontend, расположенном в develop/symfony/assets
В первую вкладку upload, под uppy widget, нужно добавить строку, построенную из config.tariff.

Пример config.tariff
"tariff": {
"title": "Free",
"delay": 3600,
"instance": 1,
"videoDuration": 3600,
"videoSize": 100,
"width": 1920,
"height": 1280,
"storage": {
"now": 0,
"max": 1073741824,
"hour": 24
}
},


Строка на английском описывает требования к загружаемому файлу.
Указывается % заполнения хранилища и данные в Мб / Гб (11 Мб из 1 Гб).
Файл должен быть не больше width x height разрешения.
Размер файла не должен быть больше videoSize Мб.
Если в хранилище осталось меньше VideoSize мегабайт, выводимое число videoSize уменьшается с пометкой "так как заканчивается место в хранилище". Всё по-английски.

Запускать сборку frontend не нужно, я тестирую в dev окружении symfony.
Проверить синтаксис можно командой node --check
